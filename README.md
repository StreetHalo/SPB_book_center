# Book Petersburg

### About

Guide to book places in St. Petersburg: bookstores, libraries and publishers. <br />
You can see information for each place.<br />
The site profile contains:<br />
-contacts (email, phone, web);<br />
-short place description;<br />
-address and working hours;<br />
-list of additional services (coffee shop, coworking, publishing your book, printing, etc.)<br />

This app using MVP pattern, clean arcitecture and Google Maps API.

### Libraries
* [ReactiveLocation](https://github.com/mcharmas/Android-ReactiveLocation) - for work with user location.<br />

For work with HTTP requests, async tasks and dependency injections:<br />
* Retrofit, RxJava and Dagger 2.<br />

Views:<br />
* [Android About Page](https://github.com/medyo/android-about-page),
[AndroidImageSlider](https://github.com/daimajia/AndroidImageSlider),
[MaterialSearchView](https://github.com/MiguelCatalan/MaterialSearchView),
[ExpandableTextView](https://github.com/Manabu-GT/ExpandableTextView).<br />
